BI SNAP (Standar Nasional Open API Pembayaran) is Indonesia’s national open API standard for digital payments, set by Bank Indonesia and managed by ASPI. It enables interoperable, secure, and standardized API-based payment methods, covering e-wallets (like OVO, DANA, ShopeePay), QRIS, bank transfers, and virtual accounts.

**Xendit fully supports BI SNAP**, allowing you to process payments via one standard API. You can follow the integration using this [API reference](https://archive.developers.xendit.co/api-reference/#bi-snap).
