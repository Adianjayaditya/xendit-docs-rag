**Payment Link** is a simple way to accept one time payment by generating a Xendit hosted page URL that customer can use to complete their purchase. You can learn more about the product capability [here](https://xendit-docs.document360.io/docs/how-payment-links-work).

You can create a Payment Link via:

- **Xendit Dashboard**
- **API integration** — [View API Docs](https://archive.developers.xendit.co/api-reference/#invoices) *(previously called* ***Invoice****)*

---

> **Heads up:** Payment Link is a legacy product. While it continues to function for existing users, we’re actively developing a new and improved experience under the [Payment Session](https://xendit-docs.document360.io/docs/how-payment-sessions-work) capability to support one time payment.
