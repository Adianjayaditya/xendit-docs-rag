# 7-Eleven CliQQ

![](https://cdn.document360.io/217abc43-8677-41fb-a81d-fceeb1fa0358/Images/Documentation/Cliqq App Logo.png?sv=2022-11-02&spr=https&st=2026-04-23T06%3A34%3A00Z&se=2026-04-23T06%3A45%3A00Z&sr=c&sp=r&sig=HV8fIRiQeiUqOaCZEXhHaFd3HgXicdVGAsEawJEIX%2Fo%3D "image(23).png")

Cliqq is a payment option available at 7-Eleven stores in the Philippines, allowing customers to pay for online purchases at any 7-Eleven outlet using cash while generating your own payment code. This method is ideal for customers without bank accounts or credit cards.

## How to pay

1. Visit your nearest 7-Eleven branch before the payment code is expired
2. Generate the payment code via 7/11 store’s kiosk or via Cliqq App make sure the biller name and transaction amount on the code is correct
3. Present the payment code to cashier to process the payment
4. Confirm the payment
5. Save the payment proof
6. Payment has successfully completed
