# Value Added Tax (VAT)

VAT (Value-Added Tax) is a consumption tax applied to goods and services at each stage of production or distribution. The VAT rate varies depending on the country of service, and for Xendit services, fees are generally **VAT-exclusive**, meaning VAT will be added to the fees as applicable.

## VAT applicability by country

Xendit applies VAT to all services in the following countries:

1. **Indonesia**

   - **VAT Rate**: 12% (The subtotal is multiplied by 11/12, bringing effective VAT rate to 11%)
2. **Malaysia**

   - **VAT Equivalent (SST)**: 0%
3. **Thailand**

   - **VAT Rate**: 7%
4. **Vietnam**

   - **VAT Rate**: 10%
5. **Philippines**

   - **VAT Rate**: 12%
6. **Singapore**

   - **VAT equivalent** **(GST)**: 0%

### VAT calculation

VAT is calculated on the **transaction fee** using the applicable VAT rate for the country where the service is provided.

In case of [Direct Deduction](/v1/docs/transaction-fees), this amount will be directly deducted from the transaction at settlement.

#### Example calculation (Vietnam)

- **Fee Rate**: VND 2,500 per transaction
- **VAT (10%)**
- **Total Fee** and **Total Fee VAT** = **2500** + **2500 × 10**% = VND 2,750

#### Example calculation (Indonesia)

- **Fee Rate**: IDR 5,000 per transaction
- **VAT (12%)**
- **Tax base = 11/12 × 5000**
- **Total Fee** and **Total Fee VAT** = **5000** + **11/12 × 5000 × 12**% = IDR 5,550
